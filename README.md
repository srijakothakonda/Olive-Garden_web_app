# 🍽️ Olive Garden – Restaurant Management & Booking Website

## 📌 Project Overview

Olive Garden is a modern restaurant management and booking website developed to simplify customer interaction with restaurants. The platform allows customers to explore menu items, reserve tables, book special events, and contact the restaurant through an easy-to-use web interface.

The project aims to improve customer convenience while helping restaurants manage bookings efficiently through a digital platform.

---

## 🚀 Features

### 🏠 Home Page

* Attractive restaurant landing page
* Easy navigation to all modules
* Restaurant introduction and highlights

### 🍕 Menu Page

* Categorized food items
* Food descriptions and pricing
* User-friendly menu browsing

### 📅 Table Booking

* Online table reservation
* Customer details collection
* Booking confirmation message

### 🎉 Event Booking

* Book parties and special events
* Select date, time, and number of guests
* Submit event requirements

### 📞 Contact Page

* Restaurant contact information
* Customer inquiry form
* Feedback submission

### ℹ️ About Us

* Restaurant history
* Vision and mission
* Business information
